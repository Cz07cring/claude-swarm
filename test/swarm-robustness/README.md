# 蜂群鲁棒性测试

## 测试目标
验证 Claude Swarm 在各种场景下的稳定性和可靠性

## 测试场景

### 1. 基础功能测试 (basic)
- 简单任务执行
- 文件创建和读取
- 基本命令执行

### 2. DAG 依赖测试 (dag)
- 线性依赖链
- 并行任务
- 复杂依赖图

### 3. 错误恢复测试 (error-recovery)
- 可重试错误（网络超时模拟）
- 不可重试错误（语法错误）
- 重试限制测试

### 4. 并发压力测试 (concurrency)
- 多任务并发执行
- 资源竞争测试
- 状态同步验证

### 5. 边界条件测试 (edge-cases)
- 空任务
- 超长任务描述
- 特殊字符处理

## 测试结果

测试结果将保存在 `results/` 目录下。
